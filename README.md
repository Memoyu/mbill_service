## 项目介绍

项目为个人记账管理系统的后台服务实现。

采用自己封装的项目模板进行快速开发，并在此项目中发现模板的问题以及需要向模板新增的功能。

部署文档：[mbill部署相关 ](https://www.yuque.com/memoyu/sz8ae0/pnbr6haohkhid0tf)
